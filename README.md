## 🇬🇧 English

PT. Yoga Wibawa Mandiri is a trusted cement packaging company strategically located at Pelabuhan Krueng Geukueh, Lhokseumawe, Aceh. This is the modern corporate website built with React, TypeScript, and Supabase.

### Features
- **Responsive Design** — Optimal on all devices
- **Modern UI/UX** — Premium design with smooth animations
- **Interactive Maps** — Factory location and distribution areas
- **Gallery** — Facility and activity documentation
- **Contact System** — Integrated contact form
- **Supabase Backend** — PostgreSQL database with real-time data
- **Company Profile** — Complete company information

### Tech Stack
- **React 18.3** + **TypeScript 5.5**
- **Vite 5.4** — Fast build tool
- **Tailwind CSS 3.4** — Utility-first CSS
- **Shadcn/UI** — High-quality components
- **Supabase** — Backend-as-a-Service
- **React Router 6** — Client-side routing
- **TanStack Query 5** — Data fetching & caching

### Quick Start

```bash
git clone https://github.com/mulkymalikuldhrs/yoga-wibawa-mandiri.git
cd yoga-wibawa-mandiri
npm install
npm run dev
```
